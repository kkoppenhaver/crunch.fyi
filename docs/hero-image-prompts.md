# Hero Image Prompts

15 retro-futuristic hero image prompts for article pages. Use with Nano Banana (Gemini) image generation.

**Color palette:** Forest green (#0a8935) + white
**Style:** 1970s/80s retro-futuristic, inspired by Loki TVA aesthetic

---

## 1. Diagonal Stripes

### 1a - Varying Widths
```
Create an abstract hero image with bold diagonal stripes in forest green (#0a8935) on a clean white background. The stripes should vary in width from thin to thick, with soft rounded ends rather than hard cuts. Style it like a 1970s poster design with subtle film grain texture throughout. The composition should be clean and graphic, feeling warm and tactile like vintage print material. Aspect ratio 16:9, high resolution.
```

### 1b - Halftone Transition
```
Create an abstract hero image with diagonal stripes in forest green (#0a8935) that transition from solid color to halftone dots as they travel across the frame. White background. The stripes should converge toward the bottom right corner, creating dynamic movement. Retro print aesthetic like a 1970s magazine advertisement. Include subtle paper texture and warm film grain. Aspect ratio 16:9, high resolution.
```

### 1c - Layered Depth
```
Create an abstract hero image with layered diagonal stripes set at slightly different angles to create depth. Forest green (#0a8935) stripes with varying opacity levels on white background. Soft shadows fall between the layers suggesting the stripes are floating on different planes. Inspired by 1970s Swiss graphic design. Clean, minimal, sophisticated. Subtle film grain texture. Aspect ratio 16:9, high resolution.
```

## Usage

Once generated, drop images into `public/hero-images/` with any filename:
- `hero-01.jpg`
- `hero-02.png`
- `my-cool-image.webp`
- etc.

Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

The system automatically scans the directory and randomly selects from available images.
Falls back to a default Unsplash image if no hero images are present.
